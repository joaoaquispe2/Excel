### :thinking: ¿Qué es Excel?

Es un programa de hojas de cálculo, usado para organizar datos en cuadrículas formadas por filas y columnas, dónde la intersección de ambas se llama celda. Permite hacer cálculos matemáticos básicos hasta el análisis de datos complejos y creación de gráficos visuales.

### :one: ¿Qué contiene este repositorio sobre Excel?

A continuación veremos los principales módulos realizados en el proceso de aprendizaje

> 1. **Fundamentals:** Conocimientos básicos de Excel como tablas definidas o formato condicional
> 2. **Formulas:** Principales fórmulas y funciones usadas en Excel
> 3. **Pivot Tables:** Tablas que permiten resumir gran cantidad de datos
> 4. **Clean, Transformation and Validation** Procesos muy relevantes dentro del análsis de datos, pues permite mejorar la calidad de los datos y su formato
> 5. **Visualization** Gráficos en Excel, y su aplicación junto con tablas dinámicas
> 6. **Automation:** Uso de macros para automatizar tareas repetitivas

Además de esto se tiene una carpeta **Projects** en dónde se guardarán los proyectos realizados por mi mismo con el fin de respaldar y probar el conocimiento que he adquirido
